# 🔐 OTP Verification System (Flask)

A secure OTP (One-Time Password) verification web application built using Flask. This project demonstrates real-world authentication features including OTP generation, email delivery, expiry handling, and user verification.

---

## 🚀 Features

- 🔐 Secure OTP generation using Python
- 📧 Email-based OTP delivery (SMTP)
- ⏳ OTP expiry system with countdown timer
- 🔁 Limited attempts for better security
- 🎨 Modern UI with responsive design

---

## 🛠️ Tech Stack

- Python
- Flask
- HTML, CSS
- SMTP (Gmail)
